# 智能配图策略引擎详细指南

> 自动判断AI生成 vs 无版权图片搜索的决策框架

---

## 核心设计原则

1. **真实性优先**：需要真实质感的场景，优先用真实摄影
2. **独特性优先**：概念性、抽象性、风格化需求，优先用AI生成
3. **版权安全**：所有推荐图片确保可商用
4. **成本效益**：免费搜索优先，AI生成为补充
5. **用户体验**：每个判断都向用户说明依据

---

## 判断维度详解

### 维度1：内容类型识别

分析illustration_spec中的描述，归类到以下类型：

| 类型 | 关键词特征 | 示例描述 |
|:-----|:-----------|:---------|
| **人物肖像** | 人脸、表情、职业形象、团队合影 | "一位年轻程序员专注地看着屏幕" |
| **职场场景** | 办公室、会议、协作、工作场景 | "团队成员在会议室讨论项目" |
| **自然风光** | 山川、海洋、森林、日出日落 | "清晨的山谷被薄雾笼罩" |
| **城市建筑** | 地标、街道、建筑外观、夜景 | "上海陆家嘴的摩天大楼群" |
| **产品展示** | 实物特写、产品使用场景 | "iPhone展示在木质桌面上" |
| **动物/植物** | 宠物、野生动物、花卉、树木 | "一只橘猫趴在键盘上" |
| **食物/美食** | 菜品、饮品、烹饪过程 | "一碗热气腾腾的拉面" |
| **抽象概念** | 时间、信息、未来、思维、连接 | "信息在数字空间中流动" |
| **创意插画** | 卡通、手绘、水彩、概念艺术 | "一个机器人正在画画的卡通场景" |
| **数据可视化** | 图表、流程图、架构图、统计 | "用户增长趋势折线图" |
| **科技/科幻** | 赛博朋克、太空、AI、未来城市 | "霓虹灯闪烁的未来都市街道" |
| **情绪氛围** | 孤独、希望、焦虑、温暖、宁静 | "一个人站在雨中的背影" |

### 维度2：真实性需求评估

| 需求等级 | 判断标准 | 典型场景 |
|:---------|:---------|:---------|
| **高** | 需要真实光影、材质纹理、物理准确性 | 人物肖像、产品展示、美食摄影 |
| **中** | 需要可信的场景氛围，但不需要完全真实 | 职场场景、城市街景 |
| **低** | 概念表达优先，真实性不重要 | 抽象概念、情绪氛围、科幻场景 |

### 维度3：风格需求评估

| 风格类型 | 判断标准 | 推荐方案 |
|:---------|:---------|:---------|
| **自然摄影风** | 要求真实、纪实、生活感 | 搜索无版权图片 |
| **商业精修风** | 要求高质量、专业布光、干净背景 | 搜索无版权图片 |
| **艺术插画风** | 手绘、水彩、油画、卡通 | AI生成（可控性强） |
| **特定数字风格** | 赛博朋克、像素风、低多边形 | AI生成 |
| **极简抽象风** | 几何图形、渐变色、符号化 | AI生成或工具生成 |
| **复古/胶片风** | 老照片质感、胶片颗粒 | 搜索无版权图片（真实胶片感AI难模拟） |

### 维度4：版权合规评估

| 场景 | 风险等级 | 建议 |
|:-----|:---------|:-----|
| 人物肖像（可识别） | 🔴 高 | 使用无版权图片或获得授权；避免AI生成真实人脸（伦理风险） |
| 品牌产品 | 🟡 中 | 使用无版权图片中的产品展示；或AI生成虚构产品 |
| 标志性建筑 | 🟡 中 | 注意建筑版权（如艾菲尔铁塔夜景有版权）；使用无版权图更安全 |
| 通用场景/自然 | 🟢 低 | AI生成或无版权图片均可 |
| 抽象概念/艺术 | 🟢 低 | AI生成最优 |

### 维度5：成本效益评估

| 方案 | 时间成本 | 经济成本 | 质量可控性 | 独特性 |
|:-----|:---------|:---------|:-----------|:-------|
| 搜索无版权图片 | 低（1-2分钟） | 免费 | 低（依赖现有素材） | 低（可能被广泛使用） |
| AI生成 | 中（3-5分钟） | 按量计费 | 高（可精确控制） | 高（独一无二） |
| 混合方案 | 中 | 混合 | 高 | 高 |

### 边界情况处理

| 边界情况 | 处理策略 |
|:---------|:---------|
| 描述同时包含人物和科技界面 | 搜索真实人物 + AI生成科技元素，后期合成 |
| 需要特定品牌产品 | 搜索无版权图片中该产品的展示；不生成带真实品牌的产品图 |
| 需要中国本土场景 | 优先摄图网（中文素材丰富）；或AI生成后审核文化准确性 |
| 用户明确要求AI生成 | 尊重用户选择，跳过智能判断 |
| 用户明确要求搜索 | 尊重用户选择，跳过智能判断 |
| 判断置信度低（50/50） | 同时提供两种方案，让用户选择 |

## 搜索关键词生成策略

### 关键词优化规则

1. **英文优先**：Unsplash/Pexels/Pixabay 英文搜索效果最佳
2. **具体化**：避免"business"，用"business meeting laptop"
3. **情感词**：添加情绪关键词提升匹配度，如 "cozy workspace"
4. **排除词**：使用负向关键词过滤不需要的结果
5. **多语言备用**：英文搜索不佳时，提供中文关键词用于摄图网

### 关键词生成模板

```
基础描述 + 场景细节 + 情绪/氛围 + 画质要求

示例：
描述：程序员深夜加班
关键词：programmer working late night, code on screen, dim lighting, focused, high quality photography

描述：团队合作
关键词：diverse team collaboration, modern office, whiteboard, brainstorming, natural light, professional photography
```

### 各网站搜索链接模板

| 网站 | 搜索链接模板 |
|:-----|:-------------|
| Unsplash | `https://unsplash.com/s/photos/{keyword}` |
| Pexels | `https://www.pexels.com/search/{keyword}/` |
| Pixabay | `https://pixabay.com/images/search/{keyword}/` |
| 摄图网 | `https://699pic.com/tupian/{keyword}.html` |

## AI生成提示词优化策略

### 提示词结构模板

```
[主体描述] + [场景/环境] + [风格/艺术方向] + [光影/氛围] + [技术参数]
```

### 按内容类型优化

| 类型 | 提示词要点 | 示例 |
|:-----|:-----------|:-----|
| 抽象概念 | 使用隐喻、视觉化语言 | "visual metaphor of information flowing like water through a digital network, luminous particles, blue and white color scheme" |
| 科技场景 | 强调界面元素、光影、材质 | "futuristic AI interface hologram, dark background, cyan glow, glass morphism, clean UI design, 8k render" |
| 情绪氛围 | 强调色彩心理学、构图 | "solitude concept, single figure on empty beach at dusk, warm orange and cool blue contrast, cinematic composition" |
| 创意插画 | 明确艺术风格、媒介 | "whimsical illustration of a robot painting on canvas, watercolor style, soft pastel colors, children's book aesthetic" |
| 封面图 | 强调视觉冲击力、留白 | "bold minimalist cover design, abstract geometric shapes, deep purple gradient, magazine quality, space for text overlay" |

## 与baoyu-article-illustrator的协作

当需要为文章批量配图时：

1. 先调用 `baoyu-article-illustrator` 分析文章结构，标记配图位置
2. 对每个配图位置，运行本策略引擎进行AI生成/搜索判断
3. 汇总所有配图方案，统一向用户展示
4. 用户确认后，分别执行AI生成或提供搜索链接

## 常见错误与避免

| 错误 | 原因 | 解决方案 |
|:-----|:-----|:---------|
| AI生成的人脸不真实 | 人物肖像应优先搜索 | 加强内容类型识别，人物类强制推荐搜索 |
| 搜索到的图片风格不统一 | 缺少风格关键词 | 在搜索关键词中加入风格限定词 |
| AI生成的文字乱码 | 当前AI在文字渲染上不稳定 | 避免在AI图中生成大量文字；或使用后期添加 |
| 产品图与实物不符 | AI虚构了产品细节 | 真实产品展示必须搜索真实图片 |
| 封面图信息过载 | 提示词过于复杂 | 封面提示词要简洁，强调主体和留白 |
