# 四层质检体系与评分标准

> 完整的质量检查清单，覆盖三种工作模式（从零创作、内容优化、内容重构）

---

## 内容评分标准（0-100分）

| 维度 | 权重 | 评分指南 | 1-10分标准 |
|:-----|:-----|:---------|:-----------|
| **Hook强度** | 25% | 开篇能否留住读者？ | 1=无聊，10=必点 |
| **价值密度** | 25% | 教学、娱乐或启发？ | 1=空洞，10=满载 |
| **平台适配** | 20% | 格式、长度、风格合适？ | 1=不适配，10=完美适配 |
| **CTA清晰度** | 15% | 下一步行动明确？ | 1=无，10=强烈引导 |
| **视觉吸引力** | 15% | 配图和排版是否吸引？ | 1=无图，10=精美 |

**门槛：低于70分不进入Phase 4，返回优化。**

---

## L1 硬性规则检查（自动扫描层）

### L1-1 禁用词扫描

全文搜索以下词汇，出现则必须替换：

| 禁用词 | 替换建议 |
|:-------|:---------|
| "说白了" | "坦率的讲"、"其实就是" |
| "意味着什么" / "这意味着" | "那结果会怎样呢"、"所以呢" |
| "本质上" | "说到底"、"其实" |
| "换句话说" | "你想想看"、"也就是说" |
| "不可否认" | 直接删掉，换成正面陈述 |
| "综上所述" / "总的来说" | 换成具体的回扣句 |
| "首先...其次...最后" | 用自然转场词替代 |
| "值得注意的是" / "不难发现" | 删掉，直接说 |
| "在当今...的时代" / "随着...的发展" | 从具体事件切入 |

### L1-2 禁用标点扫描

| 禁用标点 | 替换方式 |
|:---------|:---------|
| 冒号 ":" | 用逗号替代 |
| 破折号 "——" | 用逗号或句号替代 |
| 双引号 "" "" | 用「」替代，或直接不加引号 |

### L1-3 结构性套话扫描

- "让我们来看看..." / "接下来让我们..."
- "在当今...的时代" / "随着...的发展"
- 连续使用bullet point罗列观点（超过3个需改成散文叙述）
- 大段加粗（超过2行即过度结构化）

### L1-4 空泛表述检查

确认所有AI工具/产品都使用了具体名称，没有出现：
- "AI工具"
- "某个模型"
- "相关技术"
- "这个软件"

**通过标准：以上四项扫描零命中。**

---

## L2 风格一致性检查（模式匹配层）

### L2-1 开头检查

- [ ] 是否从一个具体的、当下的事件/场景切入？（不是宏大叙事）
- [ ] 第一句话是否让读者产生"然后呢？"的冲动？
- [ ] 有没有使用任何教科书式开头？

### L2-2 节奏与结构检查

- [ ] 是否有长短句交替？（连续3句以上句式长度相近 = 节奏呆板）
- [ ] 是否有一句话独立成段的"断裂"效果？（全文至少3次）
- [ ] 偏离主线的段落后面是否有"扣主线句"拉回来？
- [ ] 有没有用疑问句来制造节奏的刹车和转向？
- [ ] 是否避免了不必要的小标题？（方法论类除外）

### L2-3 口语化检查

- [ ] 是否使用了足够的口语化表达？（全文至少8-10个不同表达）
- [ ] 有没有论述中的故意打破？（重复强调、中途打断、省略主语）
- [ ] 有没有至少一处自嘲或承认不足？
- [ ] 标点是否用来表达情绪？（"。。。""？？？""= ="至少一种）

### L2-4 标点禁令二次确认

- [ ] 全文是否完全没有冒号、破折号和双引号？

**通过标准：L2-1全部通过，L2-2至少3/4通过，L2-3至少3/4通过，L2-4通过。**

---

## L3 内容质量检查（深度审查层）

### L3-1 观点支撑检查

- [ ] 每个核心观点是否都有具体的人/场景/细节/数据支撑？
- [ ] 有没有空泛的观点只有论断没有例证？

### L3-2 知识输出方式检查

- [ ] 知识点是否以"聊着聊着顺手掏出来"的方式呈现？
- [ ] 有没有出现"下面我来介绍""首先需要了解"这种教科书式科普？
- [ ] 引用是否自然融入论述，像"聊着聊着想起来的"？

### L3-3 文化升维检查

- [ ] 有没有至少一处从具体事件连接到更大的文化/哲学/历史参照物？
- [ ] 这个连接是否自然？

### L3-4 对立面与同理心检查

- [ ] 在讲核心观点时，是否有对对方立场的理解和承认？
- [ ] 有没有先站到读者的处境里，再给出自己的视角？

### L3-5 文章类型专项检查

| 类型 | 检查点 |
|:-----|:-------|
| 调查实验型 | 有没有"亲自下场"的叙事感？过程是否有层层递进的发现？ |
| 产品体验型 | 有没有真实的使用场景？有没有跟其他产品的自然对比？ |
| 现象解读型 | 是否有"观察→好奇→研究→升维"的推进？ |
| 工具分享型 | 有没有个人故事铺垫？效果展示是否让人"卧槽"？ |
| 方法论型 | 每节是否都落到了可执行行动？是否坦诚说明了学习成本？ |

### L3-6 逐一展示检查

- [ ] 如果涉及多个产品/案例的比较，是否用了逐一展示（每个带吐槽点评）而不是一次性罗列？

**通过标准：L3-1和L3-2必须全部通过；L3-3至L3-6中至少通过相关项。**

---

## L4 活人感终审（最终人格层）

核心问题：
> "读完这篇文章，我感觉是一个有见识的普通人在认真跟我聊一件打动他的事，还是一个AI在给我输出信息？"

### L4-1 温度感

- [ ] 文中的情绪表达是体感记忆（"我当时就愣住了""鼻子一酸"）还是知识性描述（"我感到非常震撼"）？
- [ ] 如果有人物描写，这个人物是否让你"能感觉到他的体温"？

### L4-2 独特性

- [ ] 这篇文章是否有"只有这个作者才会写出来的角度"？
- [ ] 还是换一个AI博主也能写出差不多的东西？

### L4-3 姿态检查

- [ ] 文章的语气是不是"一个有见识的普通人在认真聊一件打动他的事"？
- [ ] 有没有不自觉地滑入了"导师在教学生"或"品牌在做营销"的姿态？

### L4-4 心流检查

- [ ] 从头到尾读，有没有哪个地方注意力断掉了？需要回头理解逻辑？
- [ ] 如果有，那个地方就是需要修复的节奏问题。

**通过标准：L4-1到L4-4整体感觉"这像是真人写的"。**

---

## Mode B 专项质检（内容优化）

除了L1-L4，优化后的内容还需检查：

### 保留度检查

- [ ] 原内容的核心信息是否完整保留？
- [ ] 原内容的关键数据和事实是否准确无误？
- [ ] 用户特别要求保留的部分是否还在？

### 优化效果检查

- [ ] AI味是否降低到目标级别？
- [ ] 风格是否成功迁移到目标预设？
- [ ] 结构是否比原文更合理？
- [ ] 信息密度是否提升？

### 一致性检查

- [ ] 改写后的内容前后风格是否统一？
- [ ] 新添加的内容和保留的原内容是否衔接自然？
- [ ] 语气、人称、时态是否一致？

---

## Mode C 专项质检（内容重构）

除了L1-L4，重构后的内容还需检查：

### 重构彻底性检查

- [ ] 是否存在近义词替换式改写？（如有则必须彻底重写该段）
- [ ] 是否保留了原文的标志性表达？（如有则必须改写）
- [ ] 句式结构是否与原文明显不同？
- [ ] 段落顺序是否与原文不同？
- [ ] 比喻和修辞手法是否完全重建？

### 核心信息保留度检查

- [ ] 原文的核心论点（3-5个）是否全部保留？
- [ ] 关键论据（数据、实验、推理）是否准确无误？
- [ ] 完整论证链条是否保留？逻辑关系是否完整？
- [ ] 关键技术指标和参数是否准确？

### 独立性检查

- [ ] 文章能否脱离原文独立阅读？（读者无需看过原文也能理解）
- [ ] 是否补充了必要的前置知识和背景解释？
- [ ] 术语是否有充分解释，而非依赖原文上下文？

### 作者判断章节检查

- [ ] 「真正贡献」部分是否为原创判断，而非对原文观点的复述？
- [ ] 「证据不足」部分是否指出了原文未提及的局限性？
- [ ] 「后续验证」部分是否提出了有价值的后续研究方向？
- [ ] 三个维度是否都有实质性内容，而非泛泛而谈？
