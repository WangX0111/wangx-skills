# 微信公众号平台规范与限制

> 发布前必须了解的平台规则和最佳实践

---

## 内容限制

| 项目 | 限制 |
|:-----|:-----|
| 标题长度 | ≤ 64字 |
| 摘要长度 | ≤ 120字（不填自动截取正文前54字） |
| 正文长度 | ≤ 20000字 |
| 图片宽度 | 最大677px（建议源图900px） |
| 图片大小 | ≤ 2MB |
| 图片格式 | JPG、PNG、GIF |
| 视频长度 | ≤ 30分钟 |
| 视频大小 | ≤ 1GB |

## 封面图规范

| 类型 | 尺寸 | 比例 | 格式 |
|:-----|:-----|:-----|:-----|
| 头条封面 | 900×383px | 2.35:1 | JPG/PNG |
| 次条封面 | 200×200px | 1:1 | JPG/PNG |
| 文内配图 | 建议900px宽 | - | JPG/PNG/GIF |

## 链接限制

1. 仅 `mp.weixin.qq.com` 域名链接可直接跳转
2. 外链以脚注形式展示或转换为底部引用
3. 阅读原文链接可指向任意URL

## 排版最佳实践

```
每段 ≤ 3行
段落间空一行
关键信息 **加粗**
每 300字 一个小标题
每 300字 配 1张图
风格统一（色调、圆角、滤镜）
```

## 安全注意事项

1. **Access Token**：不要硬编码，使用环境变量
2. **AppSecret**：不要提交到代码仓库，定期更换
3. **IP白名单**：在公众号后台设置服务器IP白名单
4. **内容审核**：发布前人工审核，避免敏感内容
5. **版权合规**：引用需注明来源，配图需版权合规

## API 错误码对照

| 错误码 | 说明 | 解决方案 |
|:-------|:-----|:---------|
| `40001` | AppSecret 错误 | 检查凭证 |
| `40002` | AppID 无效 | 检查AppID |
| `40014` | Access Token 过期 | 重新获取 |
| `40164` | IP白名单错误 | 添加当前IP到白名单 |
| `45001` | 文件过大 | 压缩至≤2MB |
| `45002` | 内容过长 | 控制在20000字内 |
| `45007` | 图片尺寸不符 | 调整图片尺寸 |
