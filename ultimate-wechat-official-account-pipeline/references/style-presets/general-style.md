# 通用写作风格指南

> 适用于 formal / lifestyle / business 等通用风格的写作规范

---

## 通用原则

1. **讲人话**：避免行业黑话和学术腔，用读者能懂的语言
2. **有观点**：不模棱两可，敢于下判断
3. **有细节**：用具体场景替代抽象概念
4. **有节奏**：长短句交替，段落有呼吸感
5. **有温度**：即使是理性文章，也要让读者感受到作者的存在

---

## Formal（正式科普风）

### 适用场景
行业报告、技术深度文、科普长文、政策解读

### 核心特点
- 逻辑严密，论证充分
- 数据驱动，案例翔实
- 用词准确，但不生硬
- 有清晰的结构层次

### 写作要点
- 开头：直接抛出核心问题或关键数据
- 论述：观点 → 数据/案例支撑 → 分析 → 小结
- 结尾：总结核心发现 + 前瞻展望
- 允许使用专业术语，但首次出现需解释
- 可以加小标题（H2/H3），帮助阅读

### 禁用
- 过度口语化（"卧槽""尼玛"）
- 无依据的观点
- 情绪煽动

---

## Lifestyle（轻松生活风）

### 适用场景
生活随笔、情感故事、旅行记录、读书笔记

### 核心特点
- 温暖、细腻、有画面感
- 像朋友聊天，但不散漫
- 情感真挚，不矫情
- 细节丰富，能唤起共鸣

### 写作要点
- 开头：一个具体场景或感受
- 叙述：时间线或情感线推进
- 金句：适当点缀，不堆砌
- 结尾：留白或温暖的收束
- 多用感官描写（视觉、听觉、嗅觉）

### 推荐表达
- "那天下午的阳光特别好"
- "我突然想起来"
- "说实话"
- "你有没有过这种感觉"

---

## Business（商业分析风）

### 适用场景
商业分析、管理心得、创业复盘、行业观察

### 核心特点
- 洞察力强，能看到本质
- 案例具体，有说服力
- 建议可执行，不画饼
- 承认复杂性，不简单化

### 写作要点
- 开头：一个反常识的观察或核心判断
- 分析：现象 → 原因 → 本质 → 影响
- 案例：真实、完整、有细节
- 建议：具体、可执行、坦诚说明困难
- 结尾：核心判断的强化或开放思考

### 结构模板
```
核心判断（一句话）
  ↓
背景/现象描述
  ↓
深度分析（2-3层）
  ↓
案例佐证
  ↓
可执行建议
  ↓
总结/开放思考
```

## 跨风格通用技巧

### 开头技巧
| 类型 | 示例 |
|:-----|:-----|
| 数据冲击 | "2024年，超过80%的..." |
| 故事开场 | "那天，他站在..." |
| 提问引导 | "你有没有想过..." |
| 悬念反转 | "我面试了127个候选人，只录用了3个" |
| 场景切入 | "上周三下午，我正在..." |

### 结尾技巧
| 类型 | 适用 |
|:-----|:-----|
| 金句收尾 | 所有类型 |
| 行动呼吁 | 方法论/工具分享 |
| 开放思考 | 商业分析/现象解读 |
| 情感回扣 | 生活/情感类 |
| 预告下期 | 系列文章 |

### 段落节奏
- 每段不超过3-4行（手机端）
- 重要观点前后留白
- 长短句交替，避免连续长句
- 适当使用独句段制造停顿
